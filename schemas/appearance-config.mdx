# AppearanceConfig

`AppearanceConfig` controls the visual styling and theming of your charts. It provides comprehensive customization options including color themes, typography, borders, backgrounds, and content visibility settings.

## Type Definition

```typescript
interface AppearanceConfig {
  // Theme Selection
  theme: ThemeName;
  palette: string | null;
  
  // Border Styling
  borderStyle: ChartBorderStyle;
  borderStyleName: BorderStyleName | null;
  customBorderStyle?: VizColor;
  
  // Background Configuration
  background?: ChartBackground;
  backgroundStyle?: ChartBackgroundStyle;
  customBackgroundColor?: string;
  hasTransparentBackground?: boolean;
  hasRoundedCorners?: boolean;
  
  // Typography
  font?: ChartFont;
  textScale?: ChartTextScale;
  
  // Interaction Styling
  highlightStyle?: HighlightStyle;
  
  // Content Visibility
  shouldHideTitle?: boolean;
  shouldHideSubtitle?: boolean;
  shouldHideCaption?: boolean;
  shouldHideSourceSection?: boolean;
  shouldHideLogo?: boolean;
  
  // Series Customization
  seriesConfig?: SeriesConfigMap;
  
  // Brand Kit Integration
  brandKitConfig?: ChartBrandKitConfig;
}
```

## Core Theme Configuration

### `theme`
- **Type**: `ThemeName`
- **Required**: Yes
- **Description**: The primary visual theme for your chart

```typescript
type ThemeName = 'graphy' | 'mono' | 'pastel' | 'neon' | 'custom' | 'brandKit';
```

| Theme | Description | Best For |
|-------|-------------|----------|
| `graphy` | Default vibrant theme with rich colors | General purpose, presentations |
| `mono` | Monochromatic theme with single color family | Professional reports, minimal designs |
| `pastel` | Soft, muted color palette | Gentle presentations, wellness data |
| `neon` | Bright, high-contrast colors with glow effects | Dark backgrounds, modern UI |
| `custom` | User-defined custom colors | Brand-specific styling |
| `brandKit` | Corporate brand colors and styling | Enterprise applications |

### `palette`
- **Type**: `string | null`
- **Required**: Depends on theme
- **Description**: Theme-specific color palette selection

#### Graphy Palette Options
```typescript
type GraphyPalette = 'graphy'; // Single palette option
```

#### Mono Palette Options  
```typescript
type MonoPalette = 'blue' | 'cyan' | 'green' | 'yellow' | 'orange' | 'red' | 'pink' | 'purple' | 'grey';
```

#### Pastel Palette Options
```typescript
type PastelPalette = 'green' | 'cyan' | 'blue' | 'purple' | 'plum' | 'pink' | 'red' | 'orange' | 'yellow';
```

#### Neon Palette Options
```typescript
type NeonPalette = 'cyan' | 'pink' | 'purple' | 'red' | 'orange' | 'yellow' | 'green' | 'blue';
```

## Border Configuration

### `borderStyle`
- **Type**: `ChartBorderStyle`
- **Required**: Yes
- **Description**: Type of border around the chart

```typescript
type ChartBorderStyle = 'none' | 'gradient' | 'solid';
```

### `borderStyleName`
- **Type**: `BorderStyleName | null`
- **Required**: When using gradient or solid borders
- **Description**: Specific border color/style within the theme

#### Graphy Border Styles
```typescript
// Gradient options
type GraphyGradientBorderStyle = 'neon_pink' | 'lilac' | 'blackberry' | 'sun' | 'iceland' | 'sunset' | 'ultraviolet';

// Solid options
type GraphySolidBorderStyle = 'purple' | 'plum' | 'pink' | 'red' | 'orange' | 'yellow' | 'green' | 'cyan' | 'blue' | 'dark_purple';
```

### `customBorderStyle`
- **Type**: `VizColor`
- **Required**: No
- **Description**: Custom border color override

## Background Configuration

### `background`
- **Type**: `ChartBackground`
- **Required**: No
- **Description**: Overall chart background theme

```typescript
type ChartBackground = 'DARK' | 'LIGHT';
```

### `backgroundStyle`
- **Type**: `ChartBackgroundStyle`  
- **Required**: No
- **Description**: Specific background styling approach

```typescript
type ChartBackgroundStyle = 'solid' | 'grey' | 'tint' | 'custom' | 'transparent';
```

| Style | Description | Use Case |
|-------|-------------|----------|
| `solid` | Solid background color | Standard charts |
| `grey` | Subtle grey background | Professional documents |
| `tint` | Lightly tinted background | Soft presentation style |
| `custom` | User-defined background color | Brand-specific backgrounds |
| `transparent` | No background fill | Overlay charts, presentations |

### `customBackgroundColor`
- **Type**: `string`
- **Required**: When `backgroundStyle` is `'custom'`
- **Description**: Hex color code for custom background
- **Example**: `"#f8f9fa"`, `"#1a1a1a"`

### `hasTransparentBackground`
- **Type**: `boolean`
- **Required**: No
- **Default**: `false`
- **Description**: Override background to be transparent

### `hasRoundedCorners`
- **Type**: `boolean`
- **Required**: No  
- **Default**: `false`
- **Description**: Apply rounded corners to chart container

## Typography Configuration

### `font`
- **Type**: `ChartFont`
- **Required**: No
- **Default**: `'modern'`
- **Description**: Typography style for all chart text

```typescript
type ChartFont = 'modern' | 'fresh' | 'timeless' | 'technical' | 'brand';
```

| Font | Description | Characteristics |
|------|-------------|-----------------|
| `modern` | Clean, contemporary sans-serif | Default choice, highly readable |
| `fresh` | Friendly, approachable typography | Casual presentations, consumer-facing |
| `timeless` | Classic, professional serif | Formal reports, traditional industries |
| `technical` | Monospace, code-like appearance | Technical data, development contexts |
| `brand` | Custom brand typography | Corporate identity compliance |

### `textScale`
- **Type**: `ChartTextScale`
- **Required**: No
- **Default**: `'1'`
- **Description**: Scale factor for all text elements

```typescript
type ChartTextScale = '0.8' | '1' | '1.2' | '1.4' | '1.6' | '1.8' | '2' | '2.5' | '3' | '4';
```

| Scale | Use Case | Example Scenario |
|-------|----------|------------------|
| `0.8` | Small charts, dense layouts | Dashboard widgets |
| `1` | Standard size | Default presentation |
| `1.2`-`1.8` | Larger presentations | Conference slides |
| `2`-`4` | Very large displays | Large monitors, projectors |

## Interaction Styling

### `highlightStyle`
- **Type**: `HighlightStyle`
- **Required**: No
- **Default**: `'fade-color'`
- **Description**: How elements are highlighted during interactions

```typescript
type HighlightStyle = 'grey' | 'fade-color';
```

| Style | Description | Visual Effect |
|-------|-------------|---------------|
| `grey` | Non-selected elements turn grey | High contrast selection |
| `fade-color` | Non-selected elements fade/dim | Subtle selection |

## Content Visibility

Control which chart elements are displayed:

```typescript
const minimalAppearance: AppearanceConfig = {
  theme: 'mono',
  palette: 'blue',
  borderStyle: 'none',
  borderStyleName: null,
  
  // Hide non-essential elements
  shouldHideTitle: true,
  shouldHideSubtitle: true, 
  shouldHideCaption: true,
  shouldHideSourceSection: true,
  shouldHideLogo: true
};
```

### Visibility Options

| Property | Description | Impact |
|----------|-------------|--------|
| `shouldHideTitle` | Hide main chart title | Removes title text/editor |
| `shouldHideSubtitle` | Hide chart subtitle | Removes subtitle area |
| `shouldHideCaption` | Hide chart caption | Removes caption text below chart |
| `shouldHideSourceSection` | Hide data source info | Removes source label/link |
| `shouldHideLogo` | Hide branding logo | Removes logo from chart |

## Responsive Appearance

### Screen Size Adaptation

```typescript
function getResponsiveAppearance(screenSize: 'mobile' | 'tablet' | 'desktop'): AppearanceConfig {
  const base: AppearanceConfig = {
    theme: 'graphy',
    palette: 'graphy',
    borderStyle: 'solid',
    borderStyleName: 'blue',
    background: 'LIGHT',
    font: 'modern'
  };
  
  switch (screenSize) {
    case 'mobile':
      return {
        ...base,
        textScale: '0.8',
        borderStyle: 'none',      // Remove border on mobile
        shouldHideCaption: true,   // Hide caption on small screens
        shouldHideSourceSection: true
      };
      
    case 'tablet':
      return {
        ...base,
        textScale: '1',
        borderStyle: 'solid'
      };
      
    case 'desktop':
      return {
        ...base,
        textScale: '1.2',
        hasRoundedCorners: true
      };
  }
}
```

### Caching Theme Configurations

```typescript
// Cache computed themes
const themeCache = new Map<string, AppearanceConfig>();

function getCachedTheme(themeId: string): AppearanceConfig {
  if (!themeCache.has(themeId)) {
    themeCache.set(themeId, computeTheme(themeId));
  }
  return themeCache.get(themeId)!;
}
```
